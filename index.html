8-Stage pipeline.
Nothing skipped.

SCOUT

02b
DESIGN-SCOUT
Detects your product category. Studies reference-class design. Reviews your existing UI. Finds every flaw.
Locked design system: exact CSS variables, font pair, component specs, motion profile.
03
VALIDATE
Competitive matrix. Moat check. Honest verdict before you commit your next 20 hours.
BUILD / PIVOT / STOP
04
INTERROGATE
Max 7 surgical questions. Each unlocks a specific architecture decision. Binary options — A changes 3 files, B changes 12.
No questions about business models. Only what changes real code.
05

No "I didn't realize this would take 3 weeks" at week 3.

Nothing ships without passing every gate in sequence.
07

Builds under strict rules. No stubs. No TODOs. No hardcoded credentials. Backend before frontend. Every DB query uses prepared statements.
Shippable output. Every time.

New code becomes indistinguishable from code that's been in the project for 6 months.
Quality Gates

Four gates.
No exceptions.

CG
Completeness Gate
REQUIRED
  • Every input has a destination
  • Every button does something real
  • Every route returns something
  • Every error is handled visibly
  • At least 3 edge cases identified and handled
SG
Security Gate
REQUIRED
  • CSRF on all forms
  • PDO prepared statements on all queries
  • No hardcoded credentials anywhere
  • Auth enforced on all protected routes
  • IDOR checks on user-owned data
DG
Design Gate
REQUIRED
  • Color system compliant
  • Typography locked to brief
  • Components match spec
  • Zero slop flags
  • Mobile verified at 375px
  • Signature element present — not generic
RG
Regression Gate
REQUIRED
  • New feature didn't break old features
  • Schema changes didn't orphan queries
  • Auth system still intact
  • Navigation still works end to end

Not Inter.
Not purple.

Most AI frontend output is identical. Same Inter. Same purple gradient. Same cards. DESIGN-SCOUT fixes this permanently by studying reference-class design for your actual product category.

Reviews your existing UI against that standard. Runs a 12-dimension critique — specific, surgical, not "make it more modern."

5

Every component built against the locked brief. Zero exceptions. The DESIGN-GATE catches anything that slips through.
